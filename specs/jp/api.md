# API仕様書

このドキュメントでは、本プロジェクトで提供されるAPIの仕様について説明します。

ベースURL: `/honey-note/api`

## エンドポイント一覧

### ハチミツ (Honey)

| メソッド | パス | 説明 |
| :--- | :--- | :--- |
| GET | `/honeys` | ハチミツの一覧を取得します。 |
| GET | `/honey/{id}` | 指定したIDのハチミツ詳細を取得します。 |
| PUT | `/honey/new` | 新しいハチミツを登録します。 |
| PUT | `/honey/edit` | ハチミツの情報を更新します。 |

### 花 (Flower)

| メソッド | パス | 説明 |

| GET | `/flower/{id}` | 指定したIDの花詳細を取得します。 |
| PUT | `/flower/new` | 新しい花を登録します。 |
| PUT | `/flower/edit/{id}` | 花の情報を更新します。 |
